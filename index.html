Bienvenido a mi página web

Esta página ha sido creada como parte del trabajo de la Unidad 1. Su objetivo es mostrar los pasos necesarios para el desarrollo y publicación de una página web básica, utilizando HTML y CSS.

La informática hoy en día

La informática es una de las ramas más importantes de la tecnología actual. Gracias a ella, podemos crear herramientas digitales, páginas web, aplicaciones y sistemas que facilitan la vida cotidiana.

Este proyecto representa un ejemplo sencillo de cómo una estructura HTML y un diseño CSS pueden unirse para formar una página funcional y visualmente clara.
